# Security Policy

## Reporting a Vulnerability

Please do **not** open a public issue for security bugs. Instead, report via GitHub's private security advisory form:

**https://github.com/mahdirzv/scaffold-factory/security/advisories/new**

You can expect an acknowledgment within 7 days. Please include:

- A clear description of the issue
- Steps to reproduce
- Affected versions (tag or commit SHA)
- Suggested fix, if you have one

## Supported Versions

Only the latest tagged release receives security fixes. Pin to a tag (`@v0.4.0`) rather than `main` for reproducibility.

| Version | Supported |
|---|---|
| latest | ✅ |
| older | ❌ |
